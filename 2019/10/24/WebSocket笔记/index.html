2、基于nodejs-websocket案例

  1. npm 安装 npm install nodejs-websocket
  2. 参考npm网站代码案例
  3. 新建本地服务器端（例如：webserver.js）讲案例代码粘入
  4. 修改客户端（index.html）webSocket的端口号为服务器端口(本地：‘ws://localhost:8008/’)
  5. cmd窗口运行node webserver.js
  6. index.html即可以进行交互了
